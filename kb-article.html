Project overview

This sample knowledge base article was created for library circulation staff, based on publicly available Los Angeles Public Library policies. It showcases internal documentation designed to help customer service representatives handle patron inquiries.

Content strategy

Organization: The article follows agent workflows, moving from frequent tasks like renewals to less common situations such as lost cards or damaged materials, enabling quick reference.

Scripted responses: The article includes templated answers and prepared responses that staff can use as starting points when communicating with patrons, similar to resources created for Google support teams.

Writing and design approach

Visual hierarchy for reference:

  • HTML code tags for interface elements
  • Collapsible sections to reduce cognitive load
  • Color-coded reference boxes emphasizing important details

Style standards (based on Google's style guide):

  • Tone: conversational, empathetic, welcoming
  • Clarity: accessible reading level, active voice, information prioritized by importance
  • Formatting: sentence case headers, proper capitalization for UI elements, navigation breadcrumbs
  • Responses center customer empathy

Key features

  • Task-based workflows for typical circulation scenarios
  • Eligibility reference table for quick policy verification
  • Template email and chat responses with customizable placeholders
  • Metrics reporting guidance for escalation decisions
